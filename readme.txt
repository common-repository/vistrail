Enables Vistrail on all pages.

== Description ==

This plugin adds the required javascript for Vistrail.

Vistrail is an data analytics packages that capture user behaviour on your site (where people, where 
their mouse moves, how far they scroll). The captured data can be displayed in the form of heatmaps, 
scrollmaps and session playbacks. To use Vistrail you must open an account, once you have an account
you can create a website ID for use in this plugin. Without an account and website ID the plugin does
nothing.

For more information visit:

[Vistrail](https://www.vistrail.com/)

== Installation ==

1. Upload `vistrail` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the web property ID from Vistrail (AA-00000-000) to the settings (Admin > Settings > Vistrail)
